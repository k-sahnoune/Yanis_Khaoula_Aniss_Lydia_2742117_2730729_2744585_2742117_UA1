# Calculatrice WPF

Projet de calculatrice développée en WPF pour le cours de développement d'applications bureau.

## Fonctionnalités
- Opérations basiques (+, -, ×, ÷)
- Fonctions trigonométriques (Sin, Cos, Tan, Arcsin, etc.)
- Constantes (π, e)
- Interface moderne avec boutons arrondis
